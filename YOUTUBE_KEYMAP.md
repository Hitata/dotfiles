# YouTube keymap

Stock YouTube player shortcuts I actually use.

| Chord | Action |
|---|---|
| `<S-.>` | Increase playback speed |
| `<S-,>` | Decrease playback speed |
| `/` | Focus search |
| `c` | Toggle captions |
| `k` / space | Play / pause |
| `j` / `l` | Seek -10s / +10s |
| `f` | Fullscreen |
| `t` | Theater mode |
